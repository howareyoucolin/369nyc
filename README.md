**Install & Setup Steps**
  
1. git clone https://github.com/howareyoucolin/369nyc.git
2. cd 369nyc/
3. npm install
4. npm rebuild
5. create config.js based on config.js.sample
6. npm start
7. in root folder, node dist/main.js
